# **Color Wheel App**

## STATUS:: finished

This App was programmed on swift for artists to make it easier to pick colors for drawings. <br>

✮ The App will have 5 types of color palletes (monochromatic, analogous, complementary, triadic, and split-complementary) <br>
✮ User could copy the pallete code and use it in their drawing pograms <br>
